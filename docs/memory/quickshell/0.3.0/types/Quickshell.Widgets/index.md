Quickshell.Widgets Definitions

Bundled widgets

ClippingRectangle

Rectangle capable of clipping content inside its border.

ClippingWrapperRectangle

ClippingRectangle that handles sizes and positioning for a single visual child.

IconImage

Image component for displaying widget/icon style images.

MarginWrapperManager

Helper object for applying sizes and margins to a single child item.

WrapperItem

Item that handles sizes and positioning for a single visual child.

WrapperManager

Helper object for creating components with a single visual child.

WrapperMouseArea

MouseArea wrapper item.

WrapperRectangle

Rectangle that handles sizes and positioning for a single visual child.
