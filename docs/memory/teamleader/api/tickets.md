Tickets

Required scopes: tickets

📄️tickets.list  
\
Get a list of tickets.

📄️tickets.info  
\
Get details for a single ticket.

📄️tickets.create  
\
Create a ticket.

📄️tickets.update  
\
Update a ticket.

📄️tickets.listMessages  
\
Lists messages of a ticket.

📄️tickets.getMessage  
\
Gets the ticket message.

📄️tickets.importMessage  
\
Imports an existing message to a ticket.

📄️tickets.addReply  
\
Adds a message to a ticket.

📄️tickets.addInternalMessage  
\
Adds an internal message to a ticket.
