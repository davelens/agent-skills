Companies

Companies are legal entities, usually linked to a VAT and/or local business number.

Required scopes: companies

📄️companies.list  
\
Get a list of companies.

📄️companies.info  
\
Get details for a single company.

📄️companies.add  
\
Add a new company.

📄️companies.update  
\
Update a company.

📄️companies.uploadLogo  
\
Update the logo of a company.

📄️companies.delete  
\
Delete a company.

📄️companies.tag  
\
Add a new or existing tag to a company.

📄️companies.untag  
\
Remove a tag from a company.
