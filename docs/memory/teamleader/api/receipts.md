Receipts

Receipts are expenses for which you have a receipt, but no invoice. They can be added, approved, refused or sent to bookkeeping.

📄️receipts.add  
\
Adds a receipt.

📄️receipts.approve  
\
Approve a receipt.

📄️receipts.delete  
\
Deletes a receipt.

📄️receipts.info  
\
Get details for a single receipt.

📄️receipts.listPayments  
\
List payments for a(n incoming) receipt.

📄️receipts.markAsPendingReview  
\
Mark a receipts as pending review.

📄️receipts.refuse  
\
Refuse a receipts.

📄️receipts.registerPayment  
\
Register a payment for a(n incoming) receipt.

📄️receipts.removePayment  
\
Remove a payment from a receipt.

📄️receipts.sendToBookkeeping  
\
Send a receipt to bookkeeping for processing.

📄️receipts.update  
\
Updates a receipt.

📄️receipts.updatePayment  
\
Update a payment for a(n incoming) receipt.
