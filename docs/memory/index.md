# Memory Index

## Project Overview

Centralized manifest and authoring space for markdown-based agent skills installed to `~/.config/agents/skills`.

## Subsections

- [Bootstrap Docs Mirror](bootstrap.md) — offline markdown mirror of official Bootstrap docs, pinned by version (current: 5.3.8). Backs the `bootstrap-5` skill.
- [Teamleader Focus API Docs Mirror](teamleader.md) — offline markdown mirror of the official Teamleader Focus API docs (360 endpoints + guides). Backs the `teamleader-api` skill.
